Pinball game V2.4... Updated 2024.12.05

This is a small pinball game using plain javascript. You can control the panel using 'a' and 'd' or left and right arrow keys for left and right or click the buttons. This game might not work if your browser do not support Javascript. You earn 5 points each time you catch the ball using the panel, but if you fail to catch the ball, the game would be over. The game would restart after you press the button on the pop-out dialog.

Big thanks to Nille Cheng and his book 'Learn to Create Web Games Using Javascript'(ISBN 978-7-115-46238-1), as the concept of this game and part of the algorithm is base on his codes in the book.
